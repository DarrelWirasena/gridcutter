2. Description of Service

GridCutter is a free, browser-based image-splitting tool designed to help creators prepare photos for Instagram grid layouts. The Service supports grid layouts from 3x1 up to 3x6, with output in either 4:5 (feed-optimized) or 3:4 (profile grid-optimized) aspect ratios.

Client-side processing: All image processing occurs entirely within your web browser using the HTML5 Canvas API. No image data is transmitted to any server. GridCutter has no backend image pipeline, no cloud storage, and no image database.
